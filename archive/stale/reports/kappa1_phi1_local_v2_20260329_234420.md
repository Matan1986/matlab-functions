# Local Phi1 Construction + Kappa1 Projection (v2)

Input CSV (locked): results/switching/runs/run_2026_03_10_112659_alignment_audit/alignment_audit/switching_alignment_samples.csv

## Formulas Used
1. S_norm(I,T) = S(I,T) / S_peak(T), with S_peak(T) = max_I S(I,T)
2. Local backbone: CDF_local(I,T) = S_norm(I,T)
3. Mean normalized backbone: Sbar_norm(I) = mean_T S_norm(I,T)
4. Residual: DeltaS(I,T) = S_norm(I,T) - Sbar_norm(I)
5. Unit residual per temperature when nonzero: DeltaS_unit(I,T) = DeltaS(I,T) / ||DeltaS(I,T)||
6. Local empirical Phi1: Phi1(I) = mean_T[DeltaS_unit(I,T)] / ||mean_T[DeltaS_unit(I,T)]||
7. Projection coefficient: kappa1(T) = <DeltaS(I,T), Phi1(I)> / <Phi1(I), Phi1(I)>

Discrete inner product is evaluated on the common current grid extracted from the locked CSV only.
