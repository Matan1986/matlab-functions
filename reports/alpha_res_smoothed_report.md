## Construction

- Join on `T_K`, keep rows with finite `alpha_res` (same spirit as Agent 22B).
- **Raw:** `theta = atan2(kappa2, kappa1)`, `delta_theta` = forward difference of `unwrap(theta)` along sorted `T_K`.
- **Moving average:** `movmean` on `kappa1`, `kappa2` with window **5** (odd, endpoints shrink).
- **Savitzky–Golay:** `sgolayfilt` with frame length **7**, polynomial order **2**.
- Recompute `theta_smooth`, `delta_theta_smooth` from smoothed kappas (same atan2 / unwrap / diff recipe).
