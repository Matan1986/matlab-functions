# Relaxation ver3

`Relaxation ver3` contains the magnetic relaxation analysis workflow.

## Purpose

This module analyzes relaxation curves and supports common relaxation models, including:

- logarithmic relaxation
- stretched exponential (KWW)

## Main entry point

- `main_relexation.m`
